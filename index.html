Festify

A personalized music event discovery app with a clean, adaptable interface and thoughtful user onboarding. Designed to balance usability, customization, and trust.

Project Details

Festify is a simple and friendly app to find music events that match your taste. Users can pick their favorite artists, genres, and event types during sign-up.

The interface uses light colors and includes a brightness option for comfort. The homepage shows a personal event list, with filters to find exactly what you're looking for.

Key Features

  • Personalized Onboarding
    Quick sign-up with questions about age, location, music taste, and event types
  • Smart Discovery
    Clean homepage with a personalized feed of music events
  • Advanced Filtering
    Easy filters for genre, date, location, and ticket price
  • Comprehensive Navigation
    Menu with access to bookmarks, settings, and Spotify linking
  • Detailed Event Cards
    Event details with ticket links, and options to share or save
  • Feedback System
    Rating system: like/dislike suggestions and 5-star event feedback

WebWise Academy

An educational platform that helps developers create websites that are more accessible for people with visual impairments, featuring structured courses and hands-on tools.

Project Details

WebWise Academy includes three distinct courses to guide developers through visual accessibility:

  • Understanding Visual Impairments
    Introduction to common types of visual impairments
  • Accessible Standards and Guidelines
    WCAG standards specifically for visually impaired users
  • Assistive Technologies and Tools
    Tools like screen readers, magnifiers, alt-text, and more

Key Features

  • Three focused courses on visual accessibility
  • Color blindness simulation and contrast checking tools
  • Visual Accessibility Toolkit
  • H5P Interactive content
  • OpenAI API integration for organization information
  • Progress tracking and completion certificates

Skoupa: Lamp for Meditation

An AI-powered meditation lamp that listens to your voice and suggests personalized meditations based on your emotional state.

Project Details

"Skoupa" is an interactive meditation lamp designed to help students unwind before bed. The user speaks to the lamp, expressing how they feel, and the system uses AI-powered voice analysis to suggest matching meditation routines.

Developed in two iterations: Low-fidelity paper prototype for concept testing, and high-fidelity functional prototype with touch-sensors, light, and voice-triggered AI.

  • Designed specifically for pre-sleep stress relief

Petopia

A mobile social networking app designed to help users connect around their love for animals, report lost or found pets, and access shelter information.

Project Details

Petopia is a conceptual mobile app prototype developed as part of a university HCI course. It serves as a social platform tailored for pet lovers with intuitive interactions inspired by popular social media patterns.

Key Features

  • Custom posts and alerts for lost/found pets
  • Integrated messaging and chat interface
  • Forum section for topic-based discussions
  • Interactive maps and shelter listings
  • Personal user profiles with follow system

Design Highlights

  • User-friendly navigation with intuitive icons
  • Color-coded states and consistent layout
  • Responsive feedback through pop-ups
  • Comprehensive onboarding flows

Usability Goals

  • Memorable interface patterns
  • High user satisfaction
